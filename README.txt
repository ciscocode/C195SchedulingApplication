Java Scheduling Application
This application is a scheduling desktop application which can connect to a MySQL database to add and modify customers and appointments for a fictious business.

Author: Francisco Montenegro

IDE: IntelliJ IDEA 2022.3.1 (Community Edition)
Java SE 17.0.5
Java FX SDK 17.0.2
MySQL Connector: mysql-connector-java-8.0.31

Directions:
Open the project in IntelliJ, and run the program.
