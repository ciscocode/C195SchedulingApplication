C195 Java Performance Assessment - QAM2
This application is a scheduling desktop application which can connect to a MySQL database to add and modify customers and appointments for a fictious business.

Author: Francisco Montenegro
Contact Information: fmont14@wgu.edu
Student Application Version: 1.0
Date: February 19, 2023

IDE: IntelliJ IDEA 2022.3.1 (Community Edition)
Java SE 17.0.5
Java FX SDK 17.0.2
MySQL Connector: mysql-connector-java-8.0.31

Directions:
Open the project in IntelliJ, and run the program. 

Additional Report:
The additional report included will populate a report of Customers by Country.
Upon opening the report there is a combo box where a user can select a country from a combo box, and a table will populate customers from the selected country.
